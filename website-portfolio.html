The Ivory

A Start to finish design development passion project undertaken by Horizon as a demo product to showcase to clients.

The Challenge

Our client, a well-established fashion retailer, was struggling with an outdated e-commerce platform that was slow, difficult to navigate, and not optimized for mobile devices. This resulted in high bounce rates, low conversion rates, and declining sales.

Our Approach

We began with a comprehensive audit of the existing platform, identifying pain points and opportunities for improvement. We conducted user research to understand customer needs and preferences, and analyzed competitors to identify industry best practices.

Key Features Delivered

  • Intuitive product categorization and filtering
  • Advanced search functionality with predictive suggestions
  • High-quality product imagery with zoom capabilities
  • Personalized product recommendations
  • Streamlined checkout process with multiple payment options
  • Mobile-first responsive design

Technologies Used

React.js Node.js MongoDB Stripe API AWS Responsive Design

Finance Dashboard

A comprehensive financial management dashboard for a leading fintech company that simplified complex data visualization and improved user decision-making.

The Challenge

The client needed a way to present complex financial data in an intuitive, accessible format for users with varying levels of financial literacy. Their existing solution was cluttered, difficult to navigate, and failed to provide actionable insights.

Our Approach

We took a user-centered design approach, conducting extensive research with actual users to understand their needs, pain points, and mental models. This informed our information architecture and UI design decisions.

Key Features Delivered

  • Real-time data visualization with customizable charts
  • Personalized financial insights and recommendations
  • Simplified expense tracking and categorization
  • Goal setting and progress monitoring
  • Secure authentication and data protection
  • Cross-device synchronization

Technologies Used

Vue.js D3.js Firebase Node.js Financial APIs OAuth 2.0

Restaurant Website

A modern, responsive website for a high-end restaurant chain with an integrated online reservation system that increased bookings by 35%.

The Challenge

The restaurant chain was losing potential customers due to an outdated website that wasn't mobile-friendly and had a cumbersome reservation process. They needed a solution that would showcase their culinary offerings while streamlining the booking experience.

Our Approach

We designed a visually stunning website that highlighted the restaurant's ambiance and cuisine through high-quality imagery and immersive experiences. We integrated a seamless reservation system that reduced the booking process from multiple steps to just a few clicks.

Key Features Delivered

  • Stunning visual gallery of food and restaurant spaces
  • Real-time table availability and booking system
  • Interactive menu with dietary information
  • Integration with Google Maps for location details
  • Customer review and rating system
  • Event booking for private dining

Technologies Used

HTML5/CSS3 JavaScript PHP MySQL Booking API Google Maps API
